对象方法
abs(x) 返回数的绝对值。abs(2)结果为2，abs(-2)结果为2
ceil(x) 对数进行上舍入。ceil(-15)结果为-15；ceil(-15.6)结果为-15；ceil(15.2)结果为16；ceil(15)结果为15；
floor(x) 对数进行下舍入。floor(-15)结果为-15；floor(-15.6)结果为-16；floor(15.2)结果为15；floor(15)结果为15；
random() 返回 0 ~ 1 之间的随机数。
round(x) 把数四舍五入为最接近的整数。round(3.4)结果为3
acos(x) 返回数的反余弦值。
asin(x) 返回数的反正弦值。
atan(x) 以介于 -PI/2 与 PI/2 弧度之间的数值来返回 x 的反正切值。
atan2(y,x) 返回从 x 轴到点 (x,y) 的角度（介于 -PI/2 与 PI/2 弧度之间）。
cos(x) 返回数的余弦。
exp(x) 返回 e 的指数。
log(x) 返回数的自然对数（底为e）。
max(x,y) 返回 x 和 y 中的最高值。
min(x,y) 返回 x 和 y 中的最低值。
pow(x,y) 返回 x 的 y 次幂。pow(2,3)结果为8
sin(x) 返回数的正弦。
sqrt(x) 返回数的平方根。sqrt(9)结果为3
tan(x) 返回角的正切。
toSource() 返回该对象的源代码。
valueOf() 返回 Math 对象的原始值。
  • isMaN(NaN)
  • isFinite(x)是否是有限数
  • 数字.toFixed(n)保留n位小数，返回某数四舍五入之后保留x位小数。var num1=1204.238;num1.toFixed(2);结果为1204.24;
  • 数字.toPrecision(n)保留n位数字，返回某数四舍五入之后保留x位字符。var num1=1204.238;num1.toPrecision(5);结果为1204.2;
