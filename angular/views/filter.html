内置过滤器

转成大写：{{"name" | uppercase}}

保留俩位数：{{123.456789 | number:2}}

货币格式：{{123 | currency}}

货币格式：{{123 | currency:"¥"}}

日期格式：{{ today | date:"y-M-d hh:mm:ss.sss"}}

过滤带e的：{{["ari","lerner","likes","to","eat","pizza"] | filter:"e"}}

对象过滤器：{{[{ "name":"ari", "city":"san francisco", "favorite food":"pizza" },{ "name":"nate", "city":"san francisco", "favorite food":"indian food" }] | filter:{"favorite food":"pizza"} }}

自定义函数过滤，首字母是否为大写返回true或false：{{ ["Ari","Likes","to","travel"] | filter:isCapitalized }}

json或js对象转成字符串：{{ { "name":"ari", "city":"san francisco", "favorite food":"pizza" } | json }}

截取字符串前三个字符：{{ "San Francisco is very cloudy" | limitTo:3 }}

返回数组第一个元素：{{ ['a','b','c','d','e','f'] | limitTo:1 }}

数组进行排序：{{ [{ "name":"Ari", "status":"awake" },{ "name":"Q", "status":"sleeping" },{ "name":"Nate", "status":"awake" }] | orderBy:"name":true }}

自定义：{{ "ginger loves dog treats" | lowercase | capitalize }}
